## 🚀 What is this?  
An end-to-end intelligent platform combining:  
- 📊 Interactive HTML dashboards (analysis + storytelling)  
- 🤖 AI-powered assembly line solutions  
- 🌱 Sustainability impact insights  

Built to bridge the gap between **data insights → engineering decisions → real-world implementation**.

## 📊 Key Components  

### 🔹 NEV Analysis Dashboard  
- Global vs India EV trends  
- Market growth & penetration  
- Assembly gaps & opportunities  

### 🔹 Sustainability Dashboard  
- EV vs ICE emissions analysis  
- Carbon footprint insights  
- Energy mix impact (coal vs renewable)  

---

## 🏭 Core Problem  
EV manufacturing (especially in India) lacks:  
- Real-time defect detection  
- Predictive maintenance  
- Digital twin simulation  
- Smart supply chain intelligence  

---

## 💡 Our AI Solution (6 Modules)  

- **VisionGuard** → Defect Detection (Computer Vision)  
- **PredictaLine** → Predictive Maintenance  
- **TwinSync** → Digital Twin Simulation  
- **ChainMind** → Supply Chain Intelligence  
- **FlowAgent** → Workflow Optimization  
- **AssemblyGPT** → AI Assistant for Operations
